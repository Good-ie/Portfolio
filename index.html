I'm a DevOps Engineer based in Lagos, Nigeria, with a passion for building cloud infrastructure that works in the real world — not just in tutorials.

Featured Project 01

Nukreationz

Production-Grade QR Code Solutions Platform

A full-stack PHP application for virtual business cards and QR codes. Built with production-grade DevOps: Infrastructure as Code, automated CI/CD, containerized deployment, and secure credential management on AWS.
  • 20+ AWS resources provisioned via Terraform IaC
  • GitOps CI/CD: push to main → auto-builds, pushes to ECR, deploys to EC2
  • IAM roles & Secrets Manager — zero hardcoded credentials
  • Multi-stage Docker builds reducing image size by 40%
  • Automated database migrations & seeding in pipeline
  • Health checks with automated rollback

Featured Project 02

Blinko

AI-Powered Self-Hosted Note-Taking Platform

An open-source, self-hosted note-taking app with AI-powered RAG search. Deployed on AWS with CloudFront CDN, Route 53 custom domain, and PostgreSQL on RDS. Full Terraform IaC.
  • Next.js/TypeScript app with AI-enhanced note retrieval
  • CloudFront CDN for global content delivery with HTTPS
  • Custom domain configured via Route 53 DNS
  • PostgreSQL on RDS with automated backups
  • Full Terraform IaC for all infrastructure components
  • Docker Compose for consistent dev/prod environments
